# Maven Cycles — Executive Power BI Report

This project presents a comprehensive Power BI report built for a fictional bicycle company, Maven Cycles. The objective was to analyze business performance data and deliver actionable insights to executive stakeholders through an interactive and visually intuitive dashboard.

## Project Overview

The report provides detailed analysis of key business metrics, including sales, profitability, customer segmentation, and product performance. It includes drill-down capabilities, dynamic filtering, and a mobile-optimized layout to ensure accessibility across different devices.

## Steps and Workflow

### 1. Data Loading & Transformation
- Loaded the dataset into Power BI Desktop.
- Used Power Query Editor to apply essential data cleaning and transformations:
  - Renamed and standardized column names.
  - Corrected data types.
  - Removed unnecessary or redundant fields.
  - Ensured data consistency and accuracy.

### 2. Data Modeling
- Built relationships between tables to create a clean and efficient data model.
- Applied a star-schema-like structure for optimized performance.
- Developed DAX measures and calculated columns, including:
  - Total Sales
  - Total Profit
  - Profit Margin
  - Customer Segment Analysis
  - Category and Regional Metrics

### 3. Data Visualization
Designed interactive visuals to support executive decision-making:
- Bar and column charts
- Line graphs
- KPI cards
- Matrix tables
- Slicers for dynamic filtering and drill-downs

Enhanced interactive exploration with:
- Drill-down and drill-through options
- Cross-filtering between visuals
- Context-rich tooltips

### 4. Dashboard Design
- Built a clean, easy-to-navigate dashboard tailored for executives.
- Organized visuals logically with emphasis on readability and clarity.
- Highlighted key insights through optimal layout and visual hierarchy.

### 5. Publishing & Sharing
- Published the finalized report to Power BI Web.
- Enabled secure web access for stakeholders.
- Created a mobile-optimized view for users accessing the report via mobile devices.

### 6. Automation
- Configured scheduled data refresh to keep the report updated with the latest available data.
- Ensured that stakeholders always have access to fresh insights.

## Tools & Technologies Used
- Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)
- Power BI Web
- Power BI Mobile Layouts
- Scheduled Data Refresh

This project successfully delivered:
